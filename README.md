# tkinter-tic-tac-toe  
 tkinter based tic-tac-toe  
 
 Features currently supported  
 <ul>
 <li>local multiplayer</li>
 <li>Battle against a simple AI</li>
 <li>customizable colors(player and tab background)</li>  
 <li>counter with player colors</li>
 </ul>
